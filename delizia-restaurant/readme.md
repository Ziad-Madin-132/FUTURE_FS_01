# 🍽️ Delizia Restaurant Website

## 📌 Project Overview

Delizia is a modern, responsive restaurant website designed to deliver a premium digital experience for customers.
The website showcases the restaurant’s menu, allows table reservations, and provides essential contact information in a clean and elegant interface.

---

## 🎯 Features

* Fully responsive design (mobile, tablet, desktop)
* Modern and elegant UI/UX
* Smooth animations and transitions
* Interactive navigation with hamburger menu
* Online table reservation form
* Contact page with form and embedded map
* Clean and structured menu display

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (DOM Manipulation, Events)
* Google Fonts (Montserrat)

## ⚙️ Installation & Usage

1. Clone the repository:

```
[git clone https://github.com/your-username/delizia-website.git
](https://github.com/Ziad-Madin-132/FUTURE_FS_01/new/main/delizia-restaurant)```

2. Open the project folder:

```
cd delizia-website
```

3. Run the project:

* Simply open `index.html` in your browser

---

## 🌐 Deployment (GitHub Pages)

1. Go to repository settings
2. Navigate to "Pages"
3. Select branch `main` and root folder
4. Save and get your live website link
